# Fine-Tuning BERT on IMDB Dataset (NLP Task)

## 📌 Objective
To fine-tune a pre-trained BERT model for text classification and evaluate its performance using multiple metrics.

---

## 📊 Dataset
- IMDB Movie Reviews Dataset
- Binary classification:
  - 0 → Negative
  - 1 → Positive

---

## ⚙️ Technologies Used
- Python
- PyTorch
- Hugging Face Transformers
- Scikit-learn
- Google Colab

---

## 🔄 Workflow
Raw Data → Preprocessing → Tokenization → Model Training → Evaluation → Experiments → Comparison

---

## 🧠 Model
- bert-base-uncased
- Fine-tuned for sequence classification

---

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

---

## 🧪 Experiments
1. Full BERT Fine-Tuning
2. Frozen BERT Layers
3. Fine-tuning Last 2 Layers
